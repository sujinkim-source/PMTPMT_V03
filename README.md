# PMTPMT_V03

  PMT (Platform Mockup Tool) — 사용 가이드
  Platform Management Team | Internal Use Only

 

이 툴은 플랫폼별 스토어 에셋을 실제 화면과 유사한 형태로 미리 확인할 수 있는
내부 검수용 도구입니다. 별도 설치 없이 브라우저에서 바로 실행됩니다.
 
 
----------------------------------------------------------------
  시작하는 법
----------------------------------------------------------------
 
1. 압축 파일 안의 파일을 모두 같은 폴더에 압축 해제합니다.
2. PMTPMT_V03.html 파일을 Chrome 또는 Edge로 엽니다.
3. 메인 화면에서 사용할 툴을 선택합니다.
 
※ 인터넷 연결이 필요합니다. (폰트 로딩)
※ 파일 이름을 임의로 변경하지 마세요.

----------------------------------------------------------------
  공통 유의사항
----------------------------------------------------------------
 
  · 이 툴은 검수 및 내부 확인 전용입니다. 실제 스토어 등록에
    사용하는 공식 도구가 아닙니다.
 
  · 입력한 데이터는 브라우저를 닫으면 저장되지 않습니다.
    작업 후 반드시 ZIP 다운로드로 저장해주세요.
 
  · 이미지는 권장 규격에 맞게 준비해주시면 가장 정확하게
    미리보기됩니다.
 
  · 문의사항은 Platform Management Team으로 연락해주세요.
